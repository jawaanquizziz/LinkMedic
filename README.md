LinkMedic is a powerful VS Code extension designed to help web developers find and fix broken links and image paths in real-time.

## Features

- **Real-time Path Validation**: Automatically detects broken relative paths in HTML and PHP files.
- **Visual Cues**: Highlights problematic links with red squiggly lines directly in your editor.
- **Developer-Friendly**: Lightweight and works out of the box with no configuration needed.

## Requirements

No special dependencies are required. Just install and start coding!

## Known Issues

- Currently supports `.html` and `.php` files. Support for more languages is coming soon.
